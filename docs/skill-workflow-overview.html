WeChat Skill Workflow

微信公众号内容生产 Skill 工作流程总览

从选题到可复制 HTML：写作、审核、配图、排版和预览围绕同一份主稿推进。

输入

主题 资料 提纲 草稿 Markdown HTML
任务卡

目标读者、文章目标、口吻限制、事实边界、交付形态。

默认入口

使用 wechat-article-pipeline 编排完整流程。

Pipeline 编排器 先定策略，再生成 HTML 主稿；审核、配图和样式检查围绕同一份主稿闭环。
01

任务澄清

整理读者、目标、资料、限制和自动执行模式。

wechat-article-pipeline
02

写作策略

确定标题承诺、大纲、声口、开头和结尾动作。

wechat-auto-article-writer
03

HTML 首稿

直接生成微信友好的语义化 HTML 正文。

wechat-article-formatter
04

正文抽取

从 article#wechatArticle 抽取纯文本用于审核。

pipeline gate
05

AI 风险审核

评估 AI 痕迹、低信息密度、模板化和证据限制。

ai-writing-risk-review
06

配图调研

按段落需求找图，记录来源、授权、图注和风险。

article-image-research
07

样式审核

检查层级、图片、行内样式、复制目标和本地资源。

wechat-article-formatter
08

预览交付

生成本地 HTML 预览页，复制正文到微信后台。

preview + copy
质量门控标题承诺、信息密度、事实边界和中文语感。
风险门控AI 写作风险、图片版权、肖像权和未核验事实。
交付门控干净 HTML、Base64 图片、本地预览和微信复制。

输出

HTML 主稿

唯一正文主稿，集中在 article#wechatArticle。

本地预览页

可切换主题、编辑、批注，并复制正文。

审核摘要

AI 风险等级、主要证据、反证解释和修订建议。

配图清单

图片用途、来源、授权、风险和推荐图注。
